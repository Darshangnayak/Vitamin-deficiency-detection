# 📖 Overview

Vitamin deficiencies often produce visible symptoms on different parts of the human body such as:

- 👄 Lips
- 👅 Tongue
- 👁 Eyes
- 💅 Nails
- 🧴 Skin

Traditional diagnosis requires laboratory blood tests that are expensive, invasive, and time-consuming.

This project provides a **non-invasive**, **low-cost**, and **AI-powered** solution using **Image Processing** and **Deep Learning** to detect possible vitamin deficiencies from uploaded images.

> **Note:** This system is intended only for preliminary screening and should not replace professional medical diagnosis.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Vitamin-Deficiency-Detection.git
```

Move into the project

```bash
cd Vitamin-Deficiency-Detection
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```

---

# 🖼 Workflow

1. Upload an image.
2. Image is preprocessed.
3. CNN extracts important visual features.
4. Model predicts the vitamin deficiency.
5. Prediction is displayed on the webpage.

---

# 🎯 Supported Vitamin Deficiencies

The model is trained to identify visible symptoms associated with:

# 🤖 Deep Learning Model

The project uses a **Convolutional Neural Network (CNN)** for image classification.

### Model Workflow

```
Input Image
      │
      ▼
Convolution Layer
      │
      ▼
ReLU Activation
      │
      ▼
Pooling Layer
      │
      ▼
Flatten Layer
      │
      ▼
Fully Connected Layer
      │
      ▼
Prediction
```
